Adidas Sales Analytics Dashboard 📊

An interactive and visually appealing sales analytics dashboard built using Streamlit, Pandas, Plotly, OpenPyXL, and Pillow. This dashboard helps users explore Adidas sales data through dynamic visualizations and key business insights.

🚀 Live Demo

👉 Dashboard: https://adidas-analytics-dashboard.streamlit.app/

📌 Project Overview

The Adidas Sales Analytics Dashboard provides a comprehensive view of sales performance across different regions, products, retailers, and time periods. It enables users to analyze trends, monitor KPIs, and make data-driven decisions through interactive charts and filters.

The application is developed using Streamlit, which allows rapid deployment of Python-based data applications.

✨ Features
📈 Interactive Sales Analysis
🌍 Region-wise Sales Performance
🏪 Retailer-wise Sales Insights
👟 Product Category Analysis
